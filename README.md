# CareConnect

CareConnect is a healthcare web application designed to make medical services more accessible and affordable.  
Users can find doctors, book appointments, manage health records, track health metrics, and access health education resources.

---

## Features

- User Authentication (Login / Signup)
- Doctor Search and Appointment Booking
- Medicine Price Comparison
- Diagnostic Test Booking
- Health Records Management
- Health Tracker (Blood Pressure, Heart Rate, Blood Sugar)
- Health Education Articles
- User Profile Management

---

## Tech Stack

Frontend:
- React
- TypeScript
- TailwindCSS
- Lucide Icons

---

## Installation

Clone the repository

```bash
git clone https://github.com/vivekroy8584-sketch/careconnect.git
```

Go to project folder

```bash
cd careconnect
```

Install dependencies

```bash
npm install
```

Run the app

```bash
npm start
```

Open in browser

```
http://localhost:3000
```
